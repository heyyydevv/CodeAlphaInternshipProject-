# Stock Trading Platform Project Report

## Project Title

Stock Trading Platform

## Objective

The objective of this project is to simulate a basic stock trading system in Java where users can view market prices, buy or sell stocks, and monitor portfolio performance.

## Description

This is a console-based Java mini-project designed to demonstrate core object-oriented programming concepts and business logic. The application starts with a fixed cash balance, offers a small list of market stocks, and lets the user trade shares based on available balance and owned quantity.

The project also includes portfolio persistence through a CSV file and a simple market simulation that changes stock prices randomly. This makes the system more realistic and more complete for internship submission.

## Key Features

- Display market stock list with prices
- Buy shares if enough balance is available
- Sell shares if enough quantity is owned
- Track holdings and available cash
- Calculate total portfolio value
- Simulate stock price changes
- Save and load portfolio data

## Concepts Used

- Classes and objects
- Encapsulation
- `Map`
- Conditional logic
- Loops
- Methods
- File handling
- Exception handling

## Conclusion

The Stock Trading Platform is a strong beginner Java project that demonstrates practical programming concepts using a finance-themed simulation. It is suitable for internship submission because it combines logic, collections, user input handling, and persistent data storage in one project.
