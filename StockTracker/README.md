# Stock Trading Platform

A Java console application that simulates buying and selling stocks, managing a portfolio, and tracking total value with simple market price updates.

## Features

- View available market stocks
- Buy and sell shares
- Track cash balance and holdings
- Simulate market price changes
- Save and reload portfolio data

## Run

```bash
javac -d stock-trading-platform\\out stock-trading-platform\\src\\*.java
java -cp stock-trading-platform\\out StockTradingPlatform
```
