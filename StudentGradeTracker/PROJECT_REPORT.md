# Student Grade Tracker Project Report

## Project Title

Student Grade Tracker

## Objective

The objective of this project is to develop a Java application that helps manage and analyze student grades. The program allows users to enter student details, calculate academic performance, and display a summary report.

## Description

This is a console-based Java application developed for internship task submission. It uses an `ArrayList` to store student records dynamically. Each record contains a student's name and marks. Based on the marks entered, the program assigns a letter grade and includes the record in the summary analysis.

The application also supports saving records to a CSV file and loading them automatically when the program starts. This improves usability by preserving data between sessions.

## Key Features

- Add student names and marks
- Validate marks between 0 and 100
- Display all student records in tabular format
- Calculate average marks
- Identify highest and lowest scoring students
- Count passing and failing students
- Save records to file
- Load records automatically on startup

## Concepts Used

- Classes and objects
- Encapsulation
- `ArrayList`
- Conditional statements
- Loops
- Methods
- File handling
- Exception handling

## Expected Outcome

The program helps teachers or users quickly maintain student marks and view performance summaries in a simple format. It demonstrates core Java programming concepts in a practical mini-project.

## Conclusion

The Student Grade Tracker is a beginner-friendly yet complete Java mini-project. It fulfills the task requirements while also adding useful features like data persistence and validation, making it suitable for internship submission.
